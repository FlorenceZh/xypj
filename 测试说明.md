# 宴饮人格鉴定 H5 测试说明

## ✅ 功能更新
- 彩蛋触发方式已改为**随机2%概率**触发
- 文末二维码使用本地文件（位于 ./images/QRcode.png）
- 海报生成功能已修复

## 🚀 如何使用
1. 双击运行 `run_server.bat` 启动本地服务器
2. 在浏览器中访问自动打开的地址
3. 完成12道测试题
4. 查看结果（2%概率会触发彩蛋题目）
5. 可生成分享海报

## 📋 测试要点
1. 检查彩蛋是否能在随机情况下触发（约测试50次大概率会遇到一次）
2. 检查海报生成是否正常，是否显示二维码
3. 检查所有题目的选项是否能正常选择
4. 检查最终结果计算是否正确

## 🎯 彩蛋提示
由于彩蛋触发概率较低（2%），如果测试时未触发，这是正常现象。
可以通过查看网络请求或使用浏览器的开发者工具确认二维码图片是否正确加载（/images/QRcode.png）。

## 🔧 配置说明
- 题目内容和配置均在 `questions.js` 中
- 样式在 `style.css` 中
- 核心逻辑在 `script.js` 中
