# 宴饮人格鉴定 H5网页

## 项目简介
一个基于古希腊和中国传统宴饮文化对比的H5测试网页，通过12个场景化问题来鉴定用户的社交人格类型。

## 功能特性
- 🎯 **12道精心设计的测试题目**
- 📊 **智能评分系统**
- 🎨 **美观的界面设计**
- 🎭 **4种人格类型分析**
- 🥚 **隐藏彩蛋题目**
- 📱 **响应式设计**
- 🖼️ **生成分享海报**

## 人格类型
1. **苏格拉底式对话者**（对话型得分＞70%）
2. **孔子式礼仪家**（角色型得分＞70%）
3. **阿尔喀比亚德式多面手**（两者接近，40%-60%）
4. **庄子式逍遥者**（两者得分均低）

## 文件结构
```
/
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # 核心脚本
├── questions.js        # 题库配置
├── images/             # 图片资源文件夹
└── README.md           # 项目说明
```

## 使用方法
1. 将项目文件放置在Web服务器根目录
2. 在浏览器中打开index.html
3. 开始测试并获取你的人格鉴定结果

## 配置说明
- 题目内容、评分规则和结果类型都在`questions.js`中配置
- 使用JSON格式，便于修改和扩展
- 可以方便地增加新题目或调整现有题目

## 自定义建议
1. 替换图片占位符为你的实际图片资源
2. 根据需求调整颜色主题和字体
3. 可添加更多彩蛋题目增加趣味性
4. 考虑添加数据统计功能记录用户选择
